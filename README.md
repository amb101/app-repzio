# RepZio

## Build Setup

| WARNING: **Node <= v12.2.7 required** |
| --- |
 
```bash
# install dependencies
$ yarn install
OR
$ npm install

# serve with hot reload at localhost
$ yarn serve
OR
$ npm serve

# build for production and launch server
$ yarn build
OR
$ npm build
```
